N2T
Resolving Names Into Things
Resolves (forwards) any identifier type: ARK, DOI, URN, Handle, PMID, GRID, ISSN, etc.
Keeps identifiers persistent, forwarding users to the best known web addresses
In partnership with:
EZID,
Identifiers.org,
Internet Archive,
YAMZ Metadictionary
For any kind of named thing -- digital documents and datasets, physical museum specimens, abstract vocabulary terms, etc.

N2T is also a global ARK resolver

 

N2T works with the European Bioinformatics Institute to resolve over 700 identifier classes using prefixes. You can add yours too.
