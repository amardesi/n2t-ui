N2T

  • About N2T
Keeps names (identifiers) persistent, forwarding (resolving) them to the best known web addresses
In partnership with:
  • EZID.cdlib.org
  • Identifiers.org
  • Internet Archive
  • YAMZ Metadictionary
Any kind of name – ARK, DOI, URN, Handle, PMID, PDB, Taxon, GRID, arxiv, ISSN, ...
Any kind of thing -- web pages, data, physical specimens, vocabulary terms, living beings, groups, ...
 

N2T is also a global ARK resolver

 

N2T works with Identifiers.org to resolve over 500 prefixed identifier classes. You can add yours too.
